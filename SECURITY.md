# Security Policy

## Supported Versions

The latest stable release of UHCR is supported with security updates.

| Version | Supported |
| ------- | --------- |
| 2.0.x   | ✅         |
| < 1.2   | ❌         |

## Reporting a Vulnerability

If you discover a security vulnerability in UHCR, please report it privately.

Please include:

* A description of the vulnerability
* Steps to reproduce the issue
* The affected version(s)
* Any proof-of-concept code or screenshots (if applicable)

Do not disclose the vulnerability publicly until it has been investigated and addressed.

## Response Process

Security reports will be reviewed as quickly as possible. If the issue is confirmed, a fix will be developed and released in a future version of UHCR.

The reporter may be credited for the discovery unless they prefer to remain anonymous.
